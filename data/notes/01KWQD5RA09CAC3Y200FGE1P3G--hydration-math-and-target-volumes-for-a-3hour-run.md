title: Hydration math and target volumes for a 3‑hour run
type: fact

Planned fluid consumption numbers: use 500 ml per hour as the target during the run. For a three‑hour session that is 500 ml/hr × 3 hr = 1.5 liters on-run. Add a pre-run sip of about 200 ml to start hydrated, giving a total fluid need of about 1.7 liters for the whole outing. Because the route's single public fountain at ~60 minutes is unreliable, plan for self-supported carry with at least a small buffer above 1.7 L to avoid running low. The buffer accounts for slightly higher-than-expected sweat loss (25°C, sustained effort), slower drinking rates, or losing a bottle. These numbers will be used directly to choose flasks/bottles and the vest setup.
